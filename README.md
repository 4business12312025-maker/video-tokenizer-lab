# Video Tokenizer Lab

A research project exploring efficient video tokenization and token-based video generation for consumer hardware.

Goals:

- Learn compact video representations
- Create discrete video tokens
- Enable LLM-style prediction of video sequences
- Train on freely available compute resources
- Run on consumer GPUs such as GTX 1660-class hardware

## Roadmap

Phase 1:
Video → Tokens → Video

Phase 2:
Tokens → Transformer → Tokens

Phase 3:
Hierarchical Video Tokens

Phase 4:
Token-Based Video Generation
